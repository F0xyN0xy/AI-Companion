Required Netlify environment variables:

SUPABASE_URL          = https://xxxx.supabase.co
SUPABASE_SERVICE_KEY  = your service role key (Settings > API > service_role)
JWT_SECRET            = any long random string (openssl rand -base64 32)
GMAIL_USER            = youraddress@gmail.com
GMAIL_APP_PASSWORD    = 16-char app password from myaccount.google.com/apppasswords
APP_URL               = https://ai-companion-nova.netlify.app  (or http://localhost:8888 for local dev)

NOTE: @supabase/supabase-js is no longer needed — removed from dependencies.
      Supabase is now used ONLY as a Postgres database, not for auth.
